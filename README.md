# make-mcfunc
- Minecraft上でスクリプトを実行してピカチュウを描く

## 概要
- 任意の画像をpythonスクリプトで読み込み、Minecraft上で実行できるスクリプトを生成する
- 生成したスクリプトをMinecraft上で実行することで羊毛ブロックが配置され、元画像を表現する

## 動作環境
- Minecraft Java Edition 1.13.1
